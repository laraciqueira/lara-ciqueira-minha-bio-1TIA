# Site pessoal — Lara Karoline Souza Ciqueira

Pequeno site estático com biografia e interesses, criado para estudo.

Arquivos principais

- `INDEX.HTML` — página inicial com sua biografia, gostos pessoais e hobbies.
- `interesses.html` — página com detalhes sobre filmes, culinária e músicas favoritas.
- `styles.css` — estilos do site (responsivo, mobile-first).
- `placeholder-filme.svg`, `placeholder-comida.svg` — imagens locais de placeholder.

Como abrir localmente

No Windows (PowerShell), navegue até o diretório do projeto e execute:

```powershell
Start-Process .\INDEX.HTML
```

Ou dê dois cliques em `INDEX.HTML` no Explorer.

O que você pode editar

- Altere textos em `INDEX.HTML` e `interesses.html` para atualizar seu conteúdo.
- Substitua os arquivos `placeholder-*.svg` por imagens reais (mesmo nome ou atualize o `src`).
- Edite `styles.css` para mudar cores, fontes ou layout.

Próximos passos recomendados

- Adicionar uma página de contato.
- Inserir sua foto (coloque no diretório e atualize o `src`).
- Publicar o site no GitHub Pages para ficar online.
